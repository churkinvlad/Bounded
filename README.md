# About Bounded

The main idea of ​​this font is to make a casual version of a popular display font Unbounded.

## Available font styles

| Name       | Weight class |
| ---------- | ------------ |
| ExtraLight | 200          |
| Light      | 300          |
| Regular    | 400          |
| Medium     | 500          |
| Bold       | 700          |
| Black      | 900          |

Available also as a variable font.
